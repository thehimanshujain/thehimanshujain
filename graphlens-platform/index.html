Platform Intelligence Highlights

Graph-derived insights from 96,397 dependency edges.

⚡
● CRITICAL · Dependency Fanout
dynamic_pricing has the highest dependency fanout across all pricing workflows
5+ upstream callers with no isolation layer. Bidirectional coupling with PCC JTier creates circular failure modes. Any outage cascades to TPIS, VIS, PCC, and the incentive evaluation path simultaneously.
🔁
● CRITICAL · Retry Cascade
message-service retry behavior may create cascading failures
No dead-letter queue for Rocketman delivery failures. Retried messages re-enter the same path, amplifying load 3× on both Rocketman and Cassandra simultaneously under degradation.
🧠
● HIGH · Silent Data Dependency
machine-learning-toolkit affects runtime pricing decisions silently
ML Toolkit Airflow DAGs feed min-price floors directly to dynamic_pricing. A DAG failure causes silent price miscalculation with zero Grafana alerting — invisible to oncall until an SLA breach.
🔗
● HIGH · API Over-Coupling
pricing-control-center-ui depends on multiple tightly coupled APIs
3-layer chain: UI → PCC JTier → dynamic_pricing. Any middle-layer failure blocks the entire revenue management workflow with no anti-corruption layer between UI and backend pricing logic.
