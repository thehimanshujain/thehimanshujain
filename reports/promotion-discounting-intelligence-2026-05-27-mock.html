Glossary

Business Metrics

OD (Open Discount)
Customer promo code discount. SQL: discount_amount_loc <> 0 AND ils_discount_loc <= 0
ILS (Incentive Load Service)
Auto-applied checkout discount. SQL: ils_discount_loc > 0
NOR (Net Order Revenue)
gr - promo_spend. NA benchmark ~51% NOR%GGR.
PS%GGB
Promo Spend % of Gross Bookings. Cost efficiency metric. NA benchmark ~19%.
Purple Price
Dynamic lower price to eligible customers. Controlled via metadata.purplePriceEnabled = true. Enables show_lower_price display logic.
IPA (Incentive Price Adjustment)
Revenue opportunity from green-price vs discount optimisation. Est. $2.3M–$5.0M NA Local. 96,777 min-price deals identified.
VFM (Vendor-Funded Margin)
Merchant-funded discount. Capped at 20% of unit buy price.
GBV / GGB
Gross Bookings Value — total transaction value before refunds.

Technical Terms (IS API)

Disposition States
draft → unapproved → approved → (inactive / paused / expired)
CLOSED
Requires brand, audience, or promo code qualification constraint.
OPEN
Available to all eligible customers — no restriction qualifier.
CLOSED_MARKETING_CHANNEL
Channel-gated (email/push) without promo code.
Qualification Constraint Kinds
brand, promotionCode, audience, country, startDate, endDate, perUserRedemptionCount, redemptionCount, noPushMessaging, noMessaging
Stacking
OD + ILS applied simultaneously. discount_amount_loc <> 0 AND ils_discount_loc > 0
