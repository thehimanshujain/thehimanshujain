Naming Conventions

OD (On-Demand) Campaigns +
Campaigns with promotionCode qualification. Used when merchants want to control redemption. Naming pattern: {MARKET}_{CATEGORY}_{TYPE}_{DISCOUNT}.
ILS Campaigns +
No promo code required. Automatic attach at checkout. Higher conversion; lower merchant control.
Purple Price +
metadata.purplePriceEnabled=true. Visual pricing signal in UI. US leads at 76.2% adoption; IE/NL/BE lag below 20%.

Glossary

Business Metrics

OD Attach Rate +
% of orders where an On-Demand promo code was applied at checkout.
ILS Attach Rate +
% of orders where an Incentive-Led Sale discount was automatically applied (no code required).
Purple Price +
Visual UI signal (purple badge) indicating a promotional price. metadata.purplePriceEnabled=true in IS API.
NOR%GGR +
Net Order Revenue as % of Gross Groupon Revenue. Key profitability metric. NA target ~55-56%.
